# 🚀 Blockchain Certificate Verification System

A decentralized application (DApp) that allows issuing and verifying certificates using blockchain technology.

---

## 🔥 Features

- Issue certificates on blockchain
- Verify certificates instantly
- Tamper-proof data
- MetaMask integration
- Blockchain explorer UI

---

## 🛠 Tech Stack

- Solidity
- Hardhat
- Ethers.js
- HTML, CSS, JavaScript
- MetaMask
